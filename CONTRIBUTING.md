# Contributing

Contributions are welcome! Here's how to get started:

## Getting Started

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes and commit: `git commit -m "Add your feature"`
4. Push to your fork: `git push origin feature/your-feature-name`
5. Open a Pull Request

## Ideas for Contributions

- Add GRU or Transformer architecture for comparison
- Implement multi-step forecasting
- Add Optuna/Keras Tuner hyperparameter search
- Build a Streamlit inference dashboard
- Add unit tests for preprocessing functions

## Code Style

- Follow PEP 8
- Add docstrings to all functions
- Keep cells in the notebook clean and well-commented
